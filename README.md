# Ford Car Price Prediction 🚗

## Project Overview

This project predicts the price of a Ford car using Machine Learning and provides an interactive web application built with Streamlit.

## Features

- Predict Ford car prices
- User-friendly Streamlit interface
- Trained Machine Learning model
- Real-time predictions
- Deployed on Streamlit Cloud

## Technologies Used

- Python, Pandas, NumPy, Scikit-Learn, Joblib, Streamlit ,GitHub

## Input Features

- Year, Mileage, Tax, MPG, Engine Size, Model,Transmission,Fuel Type

## Model Performance

- R² Score: 0.84
- MAE: 1377
- RMSE: 1903
